研究計畫

本中心以創造力的基礎培育，並發展在人文、教育、科技與社會等領域之創新研究，且皆朝實踐實作的方向進行；倡導「人文創新：以人為本、在地創新、共創價值」之主張。創造力是要被實踐的，創造力的執行也正是創新的體現；因此本中心從教育的場域中思考各種創新的方法，期望從執行的成果形成力量影響、並協助政府決策。
另一方面，為迎接新科技新智慧的產業環境挑戰，亦將觸角延伸至科技領域，期望以創新研究探索科技的前瞻發展。除了教育與科技這兩個特定的場域，我們也希望將創新與創造力的思考推廣在生活的進程中，因此本中心也關注服務推廣、社會創業等創新，創造社會、人本及經濟的價值。

創造力教育領域

本領域以創造力講座為主，本講座於98年5月成立，進行創造力基礎和應用研究；推廣創造力教育和啟迪創業家精神。致力於將本校創造力教育、創業教育、藝文教育推廣出去，創造力講座著實扮演相當重要的角色。

教育創新領域

中心過去投注相當心力於教育創新，執行多項教育部補助或委託之偏鄉教育創新發展計畫，為教育部偏鄉教育之政策智囊及實踐推動主力，負責理念論述及方案框架設計；執行國民教育階段學生學習精進計畫；也與溫世仁文教基金會及看見臺灣基金會合作推動未來人才培育計畫與「GHFxEdu教育創新全球樞紐實驗計畫」案。
希望達到：全球佈局、多元聯結，培育臺灣具宏觀視野的未來人才；從臺灣出發，尋找教育創新典範，讓努力被看見、典範向全球輸出；建構平台經營與創意敘事能力，匯聚人才資源、跨域共創等目標。

社會創新領域

本領域以本校第二期邁向頂尖大學計畫拔尖研究之一的「創意設計與創新研究」計畫為基礎，以「以人為本，在地創新」為核心理念，逐步爭取到經濟部一般型學界科專「區域智慧資本三年計畫」與教育部「文化創意產業知識創造、創意加值、商品化與事業化交換樞紐計畫-實踐型智庫子計畫」。

數位創新領域

過去中心承接計畫以人文領域研究為主，近年更結合科技元素，承接數位人文計畫、前瞻無人機智慧系統計畫，中心主任劉吉軒教授也獲邀開設數位人文系列課程，讓學生理解數位人文之內涵，建構人文領域學生對數位人文領域的基本認識。
